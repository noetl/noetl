# NoETL

Automation of Microservices, Data Mesh, and MLOps orchestration.  

[![PyPI version](https://badge.fury.io/py/noetl.svg)](https://badge.fury.io/py/noetl)

## Documentation

Full documentation: **[noetl.dev](https://noetl.dev)**

## Quick Start

```bash
pip install noetl
noetl server start --init-db
noetl worker start
```

See [Quick Start Guide](https://noetl.dev/docs/getting-started/quickstart) for details.

## Development

```bash
# Bootstrap complete environment
noetl run boot

# Destroy environment
noetl run destroy
```

See [Development Guide](https://noetl.dev/docs/contributing/overview) for contributing.
