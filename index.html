NoETL APT Repository

Debian/Ubuntu package repository for NoETL CLI

Installation

Add the repository and install NoETL:

echo 'deb [trusted=yes] https://noetl.github.io/noetl jammy main' | sudo tee /etc/apt/sources.list.d/noetl.list
sudo apt update
sudo apt install noetl

Verify Installation

noetl --version

Supported Distributions

The following Ubuntu distributions are supported (replace jammy with your version):

  • Ubuntu 24.04 (Noble) - use noble
  • Ubuntu 22.04 (Jammy) - use jammy
  • Ubuntu 20.04 (Focal) - use focal

Note: Currently ARM64 architecture only.

Alternative Installation Methods

  • PyPI: pip install noetlctl
  • Cargo: cargo install noetl
  • Homebrew: brew install noetl/tap/noetl
